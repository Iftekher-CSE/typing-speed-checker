Local VS Session Storage
 Similarities:
1. Both are part of Web API and user for storing data.
2. Both store data as value against each key.
3. They store values as string.

Dissimilarities :
1. SessionStorage store data untill the browser is closed. LocalStorage stored data has no expiration.
2. SessionStorage store data is never transferred to server. LocalStorage data may transferred to server.
3. SessionStorage Data storage limit is larger than a cookie (at most 5MB). LocalStorage storage limit is the maximum amongst the two.

Global scope vs block scope
1. If a variable is declared outside all functions is considered as global scope. On the other hand a variable is declared within a function or curly braces ({}) is a block scope.
2. A variable declared globally can be accessed from any where of the code within a file. A variable declared locally, within function or within curly braces ({}), can be accessed within that function or curly braces.
3. variable declared with let and const follow this mentioned rules. but variable declared with var doesn't follow this rules, this is called hoisting.
